1. 下载 Node.js

```shell
$yum install nodejs
```

2. 查看 Node 版本

```shell
$node --version

v10.23.1
```

3. 下载 npm

```shell
$yum install npm
```

4. 查看 npm 版本

```shell
$npm --version

6.14.10
```

5. 安装 n

```shell
$npm install -g n
```

6. 安装最新版 Node

```shell
$n latest
$PATH="$PATH"
$node --version

v15.10.0
```
